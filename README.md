Engenharia de Requisitos - PUC Goiás
Este repositório contém a documentação e os artefatos desenvolvidos para a disciplina de Engenharia de Requisitos. O trabalho consistiu na aplicação de métodos de levantamento e análise para a especificação de um sistema de software.

Metodologia de Trabalho
A coleta de dados e definição do sistema foram realizadas através de:

Questionário: Aplicação de perguntas a uma profissional de psicologia para identificar as necessidades e regras de negócio do sistema.

Documentação: Redação de requisitos funcionais e não funcionais com base nas respostas obtidas.

Conteúdo do Repositório
Requisitos do Sistema: Lista de funcionalidades e restrições técnicas.

Artefatos de Elicitação: Registro das informações coletadas com a profissional da área.
